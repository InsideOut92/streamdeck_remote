# StreamDeckNavigator AddOn

Mini-Addon fuer WoW (cheat-frei), um aus deiner App erzeugte Koordinaten als User-Waypoint zu setzen.

## Installation
1. Ordner `wow-addon/StreamDeckNavigator` in deinen WoW AddOn Ordner kopieren:
   - `C:\Program Files (x86)\World of Warcraft\_anniversary_\Interface\AddOns\`
2. WoW neu starten oder `/reload` ausfuehren.
3. Im AddOn-Menue pruefen, ob `StreamDeckNavigator` aktiviert ist.

## Slash Command
- `"/sdnav <x> <y> [label]"`  
  Beispiel: `"/sdnav 45.6 32.1 Questziel"`
- `"/sdnav clear"` entfernt den User-Waypoint.

Hinweis: Das AddOn setzt nur normale Ingame-Waypoints und fuehrt keine Automatisierung aus.
